# SteveOS

SteveOS is a personal operating system and portfolio website - a multi-page Next.js 15 application with glassmorphism design showcasing projects, blog content, and services.

## Commands

- `npm run dev` - Start dev server (localhost:3000)
- `npm run build` - Production build
- `npm start` - Start production server
- `npm run lint` - Run ESLint
- `npm run format` - Format code with Prettier

## Code Style

- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS v4
- **Language**: TypeScript
- **Components**: Functional, typed interfaces
- **State**: URL-based ('nugs'), Server Components (RSC)

## Structure

- `app/`: Routes (e.g., `app/about/page.tsx`)
- `components/`: UI components
- `constants/`: Static data (site config, blog posts)
- `public/`: Static assets
